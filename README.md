# Marien's 11-Year Journey

A deeply emotional, cinematic, and romantic interactive webpage dedicated to Marien Muturi, celebrating her 11-year journey living with Type 1 Diabetes since her diagnosis on February 14, 2015.

## 🌟 Features

- **Immersive Storytelling**: 8 emotionally-crafted sections that celebrate resilience, courage, and strength
- **Cinematic Design**: Blue-themed glassmorphism effects, glowing gradients, and smooth animations
- **Interactive Elements**: Flip cards, smooth scrolling, hover effects, and engaging transitions
- **Fully Responsive**: Mobile-first design that works beautifully on all devices
- **Smooth Animations**: Framer Motion-powered transitions and floating particles
- **Emotional Typography**: Elegant serif and sans-serif font pairings for visual hierarchy

## 📖 Sections

1. **Hero Landing** - Cinematic introduction with animated particles and romantic quote
2. **Journey Timeline** - Interactive vertical timeline showing 11 years of milestones
3. **Resilience** - Powerful emotional centerpiece celebrating inner strength
4. **What Makes Marien Special** - 7 animated quality cards highlighting her best traits
5. **Love Letter** - Handwritten-style section with typewriter animation
6. **Memory Gallery** - Polaroid-style image gallery with blue glowing effects
7. **11 Reasons** - Interactive flip cards listing reasons for admiration
8. **Closing Scene** - Cinematic final section with animated stars and declaration

## 🛠️ Tech Stack

- **React 18+** - Modern UI library
- **Vite** - Lightning-fast build tool
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Powerful animation library
- **PostCSS + Autoprefixer** - CSS processing

## 🚀 Getting Started

### Prerequisites
- Node.js 16+ 
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

The development server will open automatically at `http://localhost:3000`.

## 📝 Customization

### Adding Photos to Gallery
Navigate to `src/components/Gallery.jsx` and replace placeholder data with actual image paths.

### Updating Timeline Milestones
Edit `src/components/Timeline.jsx` to add or modify timeline events.

### Changing Colors
Modify the color configuration in `tailwind.config.js` to use different shades of blue or alternative colors.

### Animation Speed
Adjust animation duration values in Framer Motion `transition` props throughout components.
